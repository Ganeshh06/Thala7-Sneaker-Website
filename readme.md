# 👟 Thala 7 - Sneaker E-Commerce Website

## Overview

Thala 7 is a modern and responsive sneaker e-commerce website built using HTML and CSS. The website showcases premium sneakers with an attractive user interface, product details, shopping cart functionality, and a checkout section.


## Live Demo
https://ganeshh06.github.io/Thala7-Sneaker-Website/


## Features

* Responsive design for Desktop, Tablet, and Mobile devices
* Modern navigation bar with search functionality
* Hero section with promotional content
* Product showcase with image gallery
* Shopping cart section
* Coupon code section
* Order summary and checkout page
* Multiple payment options (Card, Crypto, Bank)
* Smooth scrolling navigation
* Razorpay and Secure Payment integration visuals

## Technologies Used

* HTML5
* CSS3
* Font Awesome Icons

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Make sure all images are placed inside the `image` folder.
4. Open `index.html` in any web browser.
